📖 Guía de Uso del Sistema

🎯 Bienvenido

Este sistema te permite gestionar el inventario de 9 gabinetes hospitalarios de forma profesional, con alertas automáticas y control total.

✅ Agregar Artículos

  • Ve a la pestaña del gabinete deseado (Gabinete 1 a 9)
  • Completa el formulario con los datos del artículo
  • Define el stock mínimo, medio y máximo
  • Indica cuántas unidades trae cada caja
  • Haz clic en "Agregar Artículo"

📥 Registrar Entrada de Stock

  • Ve a la pestaña "Movimientos"
  • Selecciona tipo: ENTRADA
  • Elige el gabinete y artículo
  • Ingresa la cantidad en cajas y/o unidades
  • El sistema sumará automáticamente al stock

📤 Registrar Salida de Stock

  • En "Movimientos" selecciona tipo: SALIDA
  • Elige el artículo
  • Indica cuánto sale
  • El stock se restará automáticamente

🚦 Sistema de Alertas

  • 🔴 CRÍTICO = Stock igual a 0 o por debajo del mínimo
  • 🟡 BAJO = Stock entre mínimo y medio
  • 🟠 MEDIO = Stock entre medio y máximo
  • 🟢 ÓPTIMO = Stock sobre el máximo

📅 Control de Caducidad

  • ⚠️ VENCIDO = Ya pasó la fecha de caducidad
  • ⏰ POR VENCER = Vence en menos de 30 días
  • ✅ VIGENTE = Más de 30 días de vigencia

📊 Dashboard

El Dashboard muestra un resumen completo de todos los gabinetes. La tabla inferior lista todos los artículos que requieren atención inmediata (críticos, vencidos o por vencer).

📋 Órdenes de Compra

  • Crea órdenes cuando detectes productos en estado crítico
  • El sistema guarda el historial completo
  • Puedes actualizar el estado de cada orden

📞 Proveedores

Mantén actualizado el directorio de proveedores para facilitar las compras.

💾 Guardar Datos

Todos los datos se guardan automáticamente en tu navegador. No necesitas hacer nada extra.

⚠️ Importante

  • Revisa el Dashboard diariamente
  • Registra todos los movimientos inmediatamente
  • Verifica las fechas de caducidad regularmente
  • Mantén actualizados los contactos de proveedores

☁️ Configuración de Firebase

Conecta tu sistema a Firebase Firestore para guardar los datos en la nube y sincronizarlos entre dispositivos.

🔧 Credenciales del Proyecto Firebase

Ve a console.firebase.google.com → Tu Proyecto → Configuración del Proyecto → Tus Apps → Config y copia los valores.

📋 Cómo configurar Firebase — Paso a paso

1

Abre Firebase Console

Desde tu celular o computador, abre el navegador y ve a:

👉 console.firebase.google.com

Inicia sesión con tu cuenta de Gmail.

2

Crea un proyecto nuevo

Haz clic en "Agregar proyecto" → Escribe el nombre inventario-vidasana → Haz clic en "Continuar" hasta terminar (puedes desactivar Google Analytics, no es necesario).

3

Crea la base de datos

En el menú de la izquierda busca "Firestore Database" → clic en "Crear base de datos" → Elige "Comenzar en modo de prueba" → clic en "Siguiente" y luego "Listo".

💡 El modo de prueba permite leer y escribir libremente durante 30 días. Es suficiente para comenzar.
4

Registra la app web

Haz clic en el ícono ⚙️ (rueda dentada) arriba a la izquierda → "Configuración del proyecto" → baja hasta "Tus apps" → clic en el ícono </> (Web) → escribe un nombre (ej: inventario) → clic en "Registrar app".

5

Copia las credenciales

Aparecerá un bloque de código así:

const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "tu-proyecto.firebaseapp.com",
  projectId: "tu-proyecto",
  storageBucket: "tu-proyecto.appspot.com",
  messagingSenderId: "123456",
  appId: "1:123:web:abc"
};

Copia cada valor y pégalo en los campos de arriba ☝️

6

✅ ¡Conectar!

Completa los campos de arriba con los valores copiados y haz clic en "💾 Guardar y Conectar". El indicador en la esquina inferior derecha debe cambiar a ✅ Firebase Conectado.

🌐 ¿Cómo subir esta página a internet?

⭐ OPCIÓN A — GitHub + Firebase Hosting (Recomendado)

Con esta opción, cada vez que guardes cambios en GitHub, la página se actualiza automáticamente en internet. No necesitas saber programar.

GitHub 1
Crea cuenta en GitHub
Ve a github.com → "Sign up" → crea cuenta gratuita con tu email.
GitHub 2
Crea un repositorio
Haz clic en "New repository" (botón verde) → Nómbralo inventario-vidasana → márcalo como Public → clic en "Create repository".
GitHub 3
Sube el archivo index.html
En la página del repositorio haz clic en "uploading an existing file" → arrastra tu archivo index.html → clic en "Commit changes" (botón verde abajo).
Firebase 1
Activa Firebase Hosting
En Firebase Console → menú izquierdo → "Hosting" → clic en "Comenzar" → sigue los pasos hasta llegar a la opción de conectar con GitHub.
Firebase 2
Conecta GitHub con Firebase
En Firebase Hosting → "Conectar a GitHub" → autoriza el acceso → selecciona el repositorio inventario-vidasana.
Firebase 3
Configura el despliegue automático
Firebase preguntará qué carpeta publicar. Escribe / (o deja en blanco) ya que el index.html está en la raíz → haz clic en "Finalizar".
✅ Listo
¡Tu página está en internet!
Firebase te dará una dirección como inventario-vidasana.web.app. Cada vez que subas un archivo nuevo a GitHub, la página se actualiza sola en segundos.

🚀 OPCIÓN B — Netlify (La más rápida, sin GitHub)

Solo arrastra y suelta el archivo. En 30 segundos tienes un enlace para compartir.

👉 Ir a Netlify Drop → Arrastra tu index.html → ¡listo! Te da un enlace al instante.

⚠️ Desventaja: el enlace cambia cada vez que subas una versión nueva. Con GitHub+Firebase siempre es el mismo enlace.
